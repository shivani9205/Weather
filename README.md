# 🌦️ Weather App

A simple Weather Application built using HTML, CSS, and JavaScript that provides real-time weather information using a public API.

---

## 🚀 Features

- Search weather of any city
- Real-time temperature display
- Humidity information
- Wind speed details
- Responsive and clean UI
- Error handling for invalid city names

---

## 🛠️ Tech Stack

- HTML
- CSS
- JavaScript
- Weather API (OpenWeatherMap or similar)

---

## ▶️ How to Run

1. Clone this repository:
   git clone https://github.com/shivani9205/Weather.git

2. Open the project folder in VS Code

3. Run `index.html` in any browser
